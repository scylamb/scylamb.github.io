堆叠注入

堆叠注入的局限性:

  1. 并不是每一个环境下都可以执行，可能受到 API 或者数据库引擎。
  2. 在 Web 中代码通常只返回一个查询结果，因此，堆叠注入第 二个语句产生错误或者结果只能被忽略

这个就是为什么我们尝试用 union select 联合查询的原因，使用堆叠注入前，我们还需要了解数据库的相关信息才可以，如表名、列名等
